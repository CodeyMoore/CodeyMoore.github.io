Tested iOS'

It "should" work iOS 14.0 as well but I'm unable to test

A simple preference bundle thats gives users the able to -

  • SBReload
  • Enter Safe Mode
  • Reset UICache
  • Restart Bluetoothd
  • Restart SpringBoard
  • Restart BackBoardd
  • Restart MediaServerd
  • Restart CallServicesd
  • Restart NSUrlsessiond
